# Ioannis Vongelis' Website

Welcome to my personal website repository.

This project contains the files for my online portfolio and personal brand website. It is built using simple web technologies and is designed to showcase my work, projects, and contact information.

## About
This website serves as my digital presence online. It includes information about me, my portfolio, and ways to connect with me.

## Features
- Responsive design
- Portfolio page
- About page
- Contact information
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Tailwind CSS

## Project Structure
- index.html - Home page
- about.html - About page
- portfolio.html - Portfolio page
- script.js - Main JavaScript file
- photos/ - Project images
- templates/ - Reusable templates

## How to View Locally
1. Open the project folder.
2. Launch the website by opening index.html in your browser.
3. Or use a local development server if preferred.

## Deployment
This website can be published using GitHub Pages or any other static hosting service.

Feel free to reach out through the contact section on the website.
